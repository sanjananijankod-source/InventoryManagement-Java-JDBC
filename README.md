# Inventory Management System (Java + JDBC)

This project is a simple Inventory / Product Management System built using **Java**, **JDBC**, and **MySQL**.

## Features
- Add new products
- Update product quantity and price
- Delete products
- View all products
- Search products by ID
- Uses MySQL database with JDBC

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## How to Run
1. Import project into Eclipse
2. Configure MySQL connection in the DAO file
3. Create database and tables
4. Run the MainApp.java
